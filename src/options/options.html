Gemini API key

Strongly recommended. Paste a free Google AI Studio key to parse your descriptions with Gemini 2.5 Flash — fast (~1s) and reliable on shorthand notation, nested repeats, and pace zones. Google's free tier covers ~1500 parses per day.

Get a key at aistudio.google.com/apikey → Sign in → "Create API key" → copy the value starting with AIza….

If left blank, the extension falls back to Chrome's on-device Gemini Nano. The fallback is greatly inferior — it routinely mishandles standard running shorthand on anything beyond a single interval. Use a key.
